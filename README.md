# Nexa OS 🚀

Nexa OS is an AI-native operating system designed to execute tasks autonomously using intelligent agents.

## 🧠 Vision
To build the world's first OS where AI doesn't assist — it executes.

## ⚙️ Core Modules

- Memory Manager → stores and retrieves context
- Task Planner → breaks goals into tasks
- Execution Engine → executes tasks
- Agent Runtime → runs autonomous agents

## 🔄 Workflow

Intent → Planning → Execution → Memory

## 🚀 Status

- Memory system (basic) ✅
- Search + scoring system ✅
- Agent system (in progress)

Build a fully autonomous AI OS.
